## IndianPoker

c++, tcp/ip 교육을 위한 indian Poker 게임 

# 개발 목적
회사에서 필요한 c++, tcp/ip능력을 키우기 위한 간단한 포커 프로젝트 진행
# 요구 사항 
* plyaer 2명과 host 1명이 진행
* 카드는 플레이어당 1팩만 사용한다 ( 1 ~ 10)
* 인디언 포커 룰을 따른다
* 베팅은 5,000,000으로 기본베팅은 100,000으로 표현한다.
* 턴제로 진행된다
* tcp/ip 교육을 받은 후 네트워크 통신을 활용한 멀티 가능 게임을 만든다.
